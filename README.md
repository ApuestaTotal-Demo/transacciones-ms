## Installation

```bash
1. Clonar proyecto
2. npm install
3. Clonar el archivo ".env.template" y renombrarlo a ".env"
4. Cambiar las variables de entorno
5. Levantar la base de datos
6. 
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
